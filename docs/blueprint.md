# **App Name**: Convergence Tech Symposium

## Core Features:

- Event Listings: Show a listing of available events.
- Event Registration: Allow students to register for technical and non-technical events, capturing their name, college, department, email, and phone number.
- Schedule Display: Provide a day-wise timeline layout showing time slots for each event during the symposium.
- Information Hub: An area for information on the schedule, contact information, and any rules/instructions on participating.
- Animated Countdown: Visually engaging countdown timer to the event date to build anticipation.

## Style Guidelines:

- Primary color: Saturated blue-purple (#6A4AEB) to represent technology and innovation.
- Background color: Dark navy (#1A202C), a desaturated version of the primary color, creating a professional look while making content pop.
- Accent color: Cyan (#48D1CC), analogous to the primary color but brighter, for interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headers and short blocks of text, and 'Inter' (sans-serif) for body text.
- Use sharp, outline-style icons with a futuristic design to represent each event and section.
- Implement a grid-based layout with generous padding and whitespace for a clean, modern look. Mobile-first design to ensure responsiveness.
- Use subtle animations for button hovers, card transitions, and scroll-based reveals to add visual interest without overwhelming the user.
